Seu Guia Rápido: 5 Passos para Começar

Passo 1: Criação de conta e usuario

Clique em "Começar Agora" para acessar o formulário de login e criação de conta. Você precisará apenas de um nome de usuário e senha para começar a usar a plataforma.

Passo 2: Criação de responsaveis

Ao fazer login, você verá seu painel de controle. Para começar, vá até o módulo de Gerenciamento de Responsáveis e adicione duas ou mais pessoas. Depois de adicionar os responsáveis, o módulo de Gerenciamento de Tarefas será ativado. Agora você pode criar as tarefas domésticas para os responsaveis.

Passo 3: Criação de Tarefas

Após adicionar os responsáveis, o módulo de Tarefas será liberado. Crie as tarefas com nome, descrição e data de vencimento. Em seguida, atribua cada uma delas a um responsável.
